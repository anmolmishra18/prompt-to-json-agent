# ✅ Final Report - Prompt to JSON Agent

## 🧠 Overview:

This 7-day project builds a prompt interpreter that:
- Accepts design prompts
- Generates responses using a small LLM (`distilgpt2`)
- Extracts structured data (type, material, color, purpose)
- Validates it using `pydantic`
- Logs prompts and responses with timestamps
- Saves validated specs to JSON files

## 📅 Daily Breakdown:

- **Day 1**: Set up project, GitHub, virtualenv, libraries
- **Day 2**: Prompt input + response generation with HuggingFace
- **Day 3**: Logged multiple prompts into a file
- **Day 4**: Extracted structured data using basic parser logic
- **Day 5**: Implemented timestamped logging system
- **Day 6**: Full pipeline – prompt → output → extract → validate → log → save
- **Day 7**: Final cleanup, GitHub push, and report

## 🔧 Technologies Used:

- Python 3.11+
- HuggingFace Transformers (`distilgpt2`)
- Pydantic
- Git & GitHub
- VS Code
